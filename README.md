# Portfolio — Idris Bozor
CV en ligne réalisé en HTML/CSS/JavaScript vanilla,
sans framework.

## Voir le site
https://b-idris.github.io/Mon-Portfolio/

## Contenu
- Présentation et candidature en formation informatique
- Formation et expériences professionnelles
- Compétences techniques et qualités
- Section Projets (web scraper Python, à venir...)
- Coordonnées

## Technologies utilisées
- **HTML5** — structure sémantique
- **CSS3** — mise en page Flexbox, variables CSS, responsive
- **JavaScript** — menu burger mobile, dark mode avec localStorage

## Fonctionnalités
- Mode sombre / clair avec mémorisation du choix (localStorage)
- Menu burger responsive sur mobile
- Navigation fluide par ancres
- Design adapté mobile et desktop
